Add a Animated Pinterest "Pin It" Button to your images.

== Description ==

Animated Pinterest "Pin It" Button for Images is a Wordpress plugin that allows your visitors to easily share your images via Pinterest.

The plugin allows you to:

* Add hover animated button to each image.
* Disable button for selected image(s).
* Styling and positioning and animating the button.
* Customizing Pin description.
* Can add different buttons and settings to each image.
* Works on desktop and mobile devices

Note:
In lite version part of settings is disabled.
Please visit [https://pinitbutton.webwapstudio.com/](https://pinitbutton.webwapstudio.com/) to see full features and demo.

= Requirements =

* WordPress 3.3.0 or higher

== Installation ==

### Install from within WordPress

1. Visit the plugins page within your dashboard and select 'Add New'
2. Search for 'Animated Pinterest "Pin It" Button for Images'
3. Click button 'Install'
4. Activate 'Animated Pinterest "Pin It" Button for Images' from your Plugins page

### Install manually

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configuration interface can be found under `Settings - Animated "Pin It" Button For Images`.
